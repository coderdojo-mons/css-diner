Pas de panique, tu va y arriver!

Tu es sur le point d'apprendre les sélecteur CSS! Les sélecteurs sont utilisés pour sélectionner les éléments auxquels nous souhaitons appliquer des styles

Exemple 1 - Une règle CSS

p {
   margin-bottom: 12px;
}

Ci-dessus, le sélecteur "p" sélectionne tout les éléments <p> et leur applique le style "margin-bottom".

Pour jouer, saisis un sélecteur CSS dans l'éditeur ci-dessous pour sélectionner les bons objets sur la table. Si tu donnes la bonne réponse, tu avances au niveau suivant.

Survoles les objets sur la table pour voir leur code HTML.
